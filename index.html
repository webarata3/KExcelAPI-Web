セルへのアクセスはA1、B2等のラベルでOK

セルへのアクセスは、シートに対してラベル（A1やB2）もしくはインデックスを指定します。そのセルに対してtoInt、toDouble、toStr等のメソッドを使いデータを取得します。

KExcel.open("book1.xlsx").use { workbook ->
    val sheet = workbook[0]

    // セルの読み込み
    // セル名でのアクセス
    println("""B2=${sheet["B2"].toStr()}""")

    // インデックスでのアクセス
    println("println("B2=${sheet[2, 1].toInt()}");");
}

データの書き込みはもっと簡単

セルへのデータの書き込みはデータ型を意識する必要はありません。指定したセルに対して値を代入するだけです。

KExcel.open("book1.xlsx").use { workbook ->
    val sheet = workbook[0]

    // セルの書き込み
    sheet["A1"] = "あいうえお"
    sheet[3, 7] = 123

    // ファイルの書き込みも簡単に
    KExcel.write(workbook, "book2.xlsx")
}

使い方

Gradleにリポジトリーを追加してライブラリーを読み込みます。依存しているのはPOIのライブラリーだけです。

ライブラリーの準備ができたら、使いたいソースで、「import link.webarata3.kexcelapi.*」します。

dependencies {
    compile 'link.webarata3.kexcelapi:kexcelapi:0.5.1'
}
import link.webarata3.kexcelapi.*
